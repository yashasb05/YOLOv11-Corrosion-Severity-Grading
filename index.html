How It Works

01

Start Backend

Run python app.py from the project folder to start the local Flask server on port 5000.

→
02

Upload Image

Drag & drop or browse to select any metal surface image. Analysis triggers automatically.

→
03

YOLOv11 Inference

The image is sent to the backend which runs YOLOv11 (best.pt) at 640px resolution.

→
04

Results Displayed

Annotated image, severity bar, confidence gauge and per-detection cards are rendered instantly.

🚀 Quick Start

Run these two commands to get everything up:

# 1 · Install dependencies (once)
pip install flask flask-cors ultralytics opencv-python

# 2 · Start the backend server
python app.py

# 3 · Open this file in your browser
frontend/index.html
